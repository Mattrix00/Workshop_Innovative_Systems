Comandi python per plottare immagini contenuti in MNIST :

--> importare il dataset :

from keras.datasets import mnist
(x_train, y_train), (x_test, y_test) = mnist.load_data()

--> resize dei dati e normalizzazione:

x_train = x_train.astype('float32')
x_test = x_test.astype('float32')
x_train /= 255
x_test /= 255

--> plot in scala di grigi su dati normalizzati (esempio: primo elemento di x_train):

import matplotlib.pyplot as plt
plt.imshow(x_test[0], cmap='gray', vmin=0, vmax=1)

--> se i dati non vengono normalizzati : 

import matplotlib.pyplot as plt
plt.imshow(x_test[0], cmap='gray', vmin=0, vmax=255)
